\subsection{Detailed Description}
The base class for \doxyref{symbol}{p.}{classcubescript_1_1symbol} types. 

A derived \doxyref{symbol}{p.}{classcubescript_1_1symbol} object acts as an adapter between the native and CubeScript \doxyref{runtime}{p.}{namespacecubescript_1_1runtime}, performing the necessary string-type conversions and mapping operations. Symbol types are unintrusive to the native mapping's interface, meaning there are no signs of registered functions or variables in C++ being accessible from a scripting language.
