\subsection{Detailed Description}
The base class for \doxyref{construct}{p.}{classcubescript_1_1construct} types. 

A derived \doxyref{construct}{p.}{classcubescript_1_1construct} class represents a syntatical \doxyref{construct}{p.}{classcubescript_1_1construct} of the CubeScript programming language - consequently, the \doxyref{construct}{p.}{classcubescript_1_1construct} class is a critical design concept of the scripting engine. The primary functions of a \doxyref{construct}{p.}{classcubescript_1_1construct} are parsing and evaluation. The parsing function reads and interprets CubeScript syntax into an internal representation for later evaluation. The evaluation function uses the internal representation to perform an operation.

Parse CubeScript code. 

\begin{Desc}
\item[Returns:]The return value is True when parsing is complete, otherwise False is the return value.\end{Desc}
The parse method is stream-oriented; when the current function call cannot complete the parsing, because not enough input has been given yet, the function will return to the caller, who will be responsible to re-call the function when more input data is received - the parsing state is preserved among multiple function calls until parsing is complete.

When a parsing error is detected an \doxyref{exception}{p.}{classcubescript_1_1exception} of parse\_\-error type is thrown.
