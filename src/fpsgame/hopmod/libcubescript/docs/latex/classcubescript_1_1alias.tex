\subsection{Detailed Description}
The \doxyref{symbol}{p.}{classcubescript_1_1symbol} class for hosting aliases. 

An \doxyref{alias}{p.}{classcubescript_1_1alias} in the CubeScript programming language is a named string which can be executed or its stored value returned; an \doxyref{alias}{p.}{classcubescript_1_1alias} can serve as a \doxyref{variable}{p.}{classcubescript_1_1variable} or function for the script writer. Aliases are typically created and exist at \doxyref{runtime}{p.}{namespacecubescript_1_1runtime} only; the \doxyref{alias}{p.}{classcubescript_1_1alias} assignment function registers newly created aliases to the \doxyref{domain}{p.}{classcubescript_1_1domain} passed from the expression object.
